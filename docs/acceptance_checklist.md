## 2. 实现阶段基础验收

用户已确认并进入代码实现后，第一批验收建议如下：

| 编号 | 功能 | 验收标准 |
| --- | --- | --- |
| IA-01 | 编译 | MSVC + CMake 构建通过 |
| IA-02 | 设备枚举 | 可刷新并选择 MUCam 设备，状态栏可显示 SDK DLL、接口类型和可选能力 |
| IA-03 | 实时预览 | 能持续显示相机图像，状态栏右侧 FPS/分辨率/时间戳遥测持续更新，UI 不明显卡顿 |
| IA-04 | 防闪烁 | 图像刷新和窗口变化时无明显整窗闪烁 |
| IA-05 | 滚轮缩放 | 鼠标滚轮可实时缩放图像，视觉锚点稳定 |
| IA-06 | 坐标转换 | 屏幕坐标和图像坐标转换正确 |
| IA-07 | 标定 | 两点标定能生成有效比例尺 |
| IA-08 | 长度测量 | 不同缩放下同一线段测量结果一致 |
| IA-09 | 结果列表 | 测量结果可显示、删除、清空 |
| IA-10 | 项目保存 | 保存再打开后标定、测量、染料资料和通道配置可恢复 |
| IA-11 | CSV 导出 | 测量结果可导出为 CSV |
| IA-12 | 图片导出 | 可导出原图或带测量叠加图 |
| IA-13 | 诊断报告 | 可保存包含 SDK、设备、帧、标定和处理状态的现场诊断报告 |

## 3. 高级图像处理验收

| 编号 | 功能 | 验收标准 |
| --- | --- | --- |
| IP-01 | 伪彩 | 伪彩只改变显示图，不修改原始灰度数据 |
| IP-02 | 荧光融合 | 2-4 个通道可设置颜色、强度范围、可见性并融合 |
| IP-03 | 染料管理 | 可新增、编辑、删除自定义染料资料 |
| IP-04 | 图像拼接 | 多张重叠图可生成全景派生图像 |
| IP-05 | EDF | Z 序列图可生成景深扩展合成图和焦点图 |
| IP-06 | 后台任务 | 拼接和 EDF 运行时 UI 不阻塞，可显示进度 |
| IP-07 | 数据追溯 | 派生图像能追溯源图和处理参数 |

## 4. 暂不验收内容

以下内容默认不属于第一阶段验收：

- Qt 迁移。
- PDF/报告导出。
- 电动载物台自动扫描。
- 电动 Z 轴联动采集。
- GPU 加速。
- 预置完整荧光染料库。
